## Starting JupyterLab

With your [virtual environment active](/software-guide/#activate-the-virtual-environment) and dependencies installed, start
JupyterLab from the project root:

```bash
jupyter lab
```

JupyterLab will open automatically in your default browser at
`http://localhost:8888`. The terminal window is running the server that the
browser connects to.
