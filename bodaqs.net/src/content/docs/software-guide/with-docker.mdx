## Install Docker

If you don't have Docker installed, follow the instructions for your operating system on the [Docker website](https://docs.docker.com/get-docker/).

Once it's installed, you can verify that it's working by running:

```bash
docker --version
```

## Get the repository


### Via git (recommended)
Pull down the BODAQS repository if you haven't already:

```bash
git clone --depth 1 git@github.com:benconnor1972/BODAQS.git
```

### As a zip file
you can also download the repository HEAD as a zip file from GitHub and extract it to your desired location:
<LinkButton iconPlacement={"start"} icon={"cloud-download"} variant="minimal" href={"https://github.com/benconnor1972/BODAQS/archive/refs/heads/main.zip"}>
  Download the repository source from GitHub
</LinkButton>

## Start the container
From the root of the repository:

```bash
cd docker
docker compose up
```
The first time you run this command, Docker will build the container image.
This may take a few minutes while the image is downloaded and all the python dependencies are installed.
Once it's done, the container will start and you should see output in the terminal indicating that
the JupyterLab server is running.
