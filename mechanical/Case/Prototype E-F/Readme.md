### 3d printing files for case

This directory contains the STL files needed to print the case. A case has four parts: base, lid, top end and bottom end.

There are two base variants, one that uses heat-set brass inserts for closure, and one that uses standard M3 hex nuts.

The ends vary according to the number and type of connectors to be installed. The standard layout is:
 - Four connectors in the top end: two analog inputs, one I2C input and one connector for a handlebar-mounted switch.
 - Two connectors in the bottom end for analog inputs.
 
Variations of the end pieces are provided should you not want to break out all the connectors.

Unless otherwise noted, the end pieces are designed for M8 circular connectors. Those designated '_plastic' are designed to accommodate push-together M8 plastic connectors with cable tails attached.

CAD files (Fusion archive and STEP) are also provided to allow modification of the case design.
